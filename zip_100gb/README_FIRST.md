This directory is the verified recovery copy for the Boichitro-MoE experiment.
Do not place it inside a folder that you plan to delete.

- Use `boichitro-moe-github-source-7cccbd8.zip` for a small, clean source copy.
- Use `boichitro-moe-github-history-7cccbd8.bundle` to recover the Git repository.
- Keep `boichitro-moe-complete-backup-2026-07-24.tar.zst` to preserve all
  datasets, checkpoints, runs, caches, reports, and the original project.
- Read `RESTORE_INSTRUCTIONS.md` for verification and restore commands.

The complete archive is too large for a normal GitHub upload. Keep it on local
or external backup storage. Before deleting the original experiment, copy this
entire directory to a second physical disk or trusted cloud storage if possible.
